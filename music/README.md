# Music bed

Drop one audio file here (mp3/wav/ogg/m4a/flac) and `render.py` will loop it
under the SFX at 30% volume automatically. No file = SFX only.

Recommended source: [NoCopyrightSounds](https://ncs.io) - free, high-energy,
copyright-safe. Credit the track in the video description, e.g.:

```
Song: Lensko - Circles [NCS Release]
Music provided by NoCopyrightSounds
Free Download/Stream: http://ncs.io/circles
```

Music files are gitignored - they don't belong in the repo.
