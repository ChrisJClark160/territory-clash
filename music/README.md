# Music bed

Drop one audio file here (mp3/wav/ogg/m4a/flac) and `render.py` will loop it
under the SFX at 30% volume automatically. No file = SFX only. If several
files exist, the first in sorted order is used.

**Name the file with the attribution built in** - describe.py puts the
filename straight into the video description as the music credit:

```
Raving Energy - Kevin MacLeod (incompetech.com CC BY 4.0).mp3
-> "Music: Raving Energy - Kevin MacLeod (incompetech.com CC BY 4.0)"
```

Current bed (2026-07-18): Raving Energy by Kevin MacLeod, 140 BPM EDM,
licensed CC BY 4.0 from incompetech.com (direct mp3 downloads, no account).

Other safe sources: [NoCopyrightSounds](https://ncs.io) (free, requires
site click-through + credit line). FreePD.com closed in 2026 - don't chase
links to it.

Music files are gitignored - they don't belong in the repo.
